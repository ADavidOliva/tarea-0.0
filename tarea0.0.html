Tarea

Al iniciar sesión de un usuario atraves del correo electronico y su contraseña, para cpger su correo y mandar un codigo de validacion.

Verificar las credenciales: Al enviar el formulario de inicio de sesión, verificar que el correo y la contraseña sean correctos mediante una consulta a la base de datos. Generar un código de validación: Si las credenciales son válidas, generar un código aleatorio. Enviar el código al correo: Enviar el código de validación al correo del usuario. Verificar el código: El usuario deberá introducir el código recibido y se validará en el servidor. Si el código es correcto, el acceso se permitirá.
